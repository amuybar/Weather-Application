## Getting Started

### Backend

See the [backend README](./backend/README.md) for full setup instructions.

```bash
cd backend
cp .env.example .env
# Add your OPENWEATHERMAP_API_KEY in .env
php artisan serve
```

### Frontend

See the [frontend README](./frontend/README.md) for full setup instructions.

```bash
cd frontend
npm install
npm run dev
```
